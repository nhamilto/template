included README¶

Acoustic tomography of the atmosphere

Getting Started¶

download the code repository from github: https://github.nrel.gov/nhamilto/ATom perform local editable installation: pip install -e .

Required packages¶

numpy - used to structure data into arrays, perform basic operations, etc. pandas - used for DataFrame stuctures, helps with data organization matplotlib - used in flow visualization

Prerequisites¶

Python 3.6 - working knowledge

Installing¶

Download or clone git repo: https://github.nrel.gov/nhamilto/ATom Go to town.
